Data Blueprint

AI Decision App: Predictive Sales & Demand Intelligence

1. Introduction

This document outlines the data schema required to power the Predictive Sales & Demand Intelligence AI model. A robust feature set, combining internal data with external signals and Axyo's automated feature engineering, is the foundation for building a highly accurate and explainable forecasting solution.

The target variable for this model is net_sales_amount, predicted on a weekly basis over a rolling 12-month horizon.

2. Internal Data Features (Client/Partner Systems)

Feature Name Description Data Type
transaction_idUnique identifier for each sales transaction.String
timestampDate and time of the transaction.Timestamp
skuStock Keeping Unit identifier.String
product_familyThe product family or category for the SKU.String
sales_regionThe sales region where the transaction occurred.String
units_soldNumber of units sold in the transaction.Integer
list_priceThe standard list price per unit at the time of sale.Float
promo_flagBoolean indicating if the sale was part of a promotion.Boolean

3. External Data Signals (Fused by Axyo)

Feature Name Description Data Type
pmi_scorePurchasing Managers' Index, a key economic health indicator.Float
copper_indexMarket price index for copper, a critical raw material.Float

4. Axyo-Engineered Features (Derived & Aggregated)

This is where raw data is transformed into predictive intelligence. These features are automatically generated by the Axyo platform.

Feature Name Description Data Type
rolling_avg_sales_90d90-day rolling average of sales for a given SKU/region.Float
days_since_last_promoNumber of days since the last promotion for a specific SKU.Integer
price_elasticity_coeffCalculated coefficient representing how demand reacts to price changes.Float
lagged_copper_index_6wThe value of the copper index from 6 weeks prior to capture delayed impact.Float
pmi_trend_3m3-month trend (slope) of the PMI score to capture economic momentum.Float
is_holiday_seasonBoolean flag indicating if the date falls within a key holiday period.Boolean
